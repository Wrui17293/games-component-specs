# games-component-specs

营销互动首页可复用 Zero 组件规范。

用于低保真布局、高保真重建和组件复用时，查询顶部通栏、常驻功能信息区、京豆、红包、资源位 Icon、进度激励区和 Feeds 的链接、尺寸、状态与使用规则。

## 文件

- `SKILL.md`：组件清单、Zero 链接和复用规则。
- `references/rules.md`：组件复用、缩放、裁切、圆角和布局边界规则。

## 流程位置

这是辅助规范，不作为独立流程阶段。通常由 `games-homepage-layout`、`games-lofi` 或 `games-hifi` 按需引用。
